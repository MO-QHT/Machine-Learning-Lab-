# Lab 7 – Logistic Regression

---

## Overview

In this lab, a Logistic Regression model was developed to predict whether a user will click on an advertisement. The prediction is based on a set of behavioral and demographic features.

---

## Work Performed

The following steps were completed during this lab:

- Imported and explored the dataset  
- Conducted basic exploratory data analysis (EDA), including:
  - Visualizing age distribution using histograms  
  - Using jointplots to study relationships between variables  
  - Creating pairplots to observe class separation based on the target variable  

- Selected important features for modeling:
  - Daily Time Spent on Site  
  - Age  
  - Area Income  
  - Daily Internet Usage  
  - Gender (Male)  

- Split the dataset into training and testing sets  
- Trained a Logistic Regression model  
- Generated predictions on the test data  

- Evaluated the model performance using:
  - Confusion Matrix  
  - Classification metrics (Precision, Recall, F1-score)  

---

## Results

- The model achieved an accuracy of approximately 91%  
- Clear separation between the classes was observed based on user behavior features  

---

## Notes

- The dataset did not require cleaning as it contained no missing values  
- Logistic Regression performed well due to clear patterns in the data
