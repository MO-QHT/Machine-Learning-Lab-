# Lab 1: Getting Started with Git and GitHub

## Objective
To learn the basics of Git and GitHub by creating a repository, making commits, and managing changes.

## What I Have Learned
- How to create and set up a GitHub account.
- How to create a repository and initialize it with a `README.md` file.
- How to edit files and commit changes on GitHub.
- How to create and use branches safely.
- How to open and merge pull requests.
- Basic GitHub workflow and version control concepts.

## Outcome
This lab provided a foundational understanding of Git and GitHub for future coursework.
