# Lab 10 — Support Vector Machines (SVM)

This repository presents the implementation of a Support Vector Machine (SVM) classification model using the Iris dataset.

---

## 🔍 Overview

The notebook includes several machine learning steps, including:

- Loading and exploring the Iris dataset
- Visualizing the dataset using:
  - Pairplots
  - KDE visualizations
- Preparing and preprocessing the data
- Splitting the dataset into training and testing sets
- Building an SVM classification model
- Generating predictions using the trained model
- Evaluating model performance using:
  - Confusion Matrix
  - Classification Report
- Performing hyperparameter tuning with `GridSearchCV`
- Comparing model results before and after tuning

---

## ⚙️ Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 🎯 Objective

The purpose of this lab is to understand how Support Vector Machines are applied in classification problems and to evaluate how tuning model parameters can improve classification accuracy and overall performance.
